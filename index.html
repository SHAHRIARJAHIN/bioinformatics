‚úÇÔ∏è 5. Read Trimming

Option A: Trimmomatic

conda create --name trimmomatic_env
conda activate trimmomatic_env
conda install -c bioconda trimmomatic

Single-end

trimmomatic SE input.fastq output_trimmed.fastq LEADING:3 TRAILING:3 SLIDINGWINDOW:4:20 MINLEN:36

Paired-end

trimmomatic PE input_1.fastq input_2.fastq output_1_paired.fastq output_1_unpaired.fastq output_2_paired.fastq output_2_unpaired.fastq ILLUMINACLIP:TruSeq3-PE.fa:2:30:10:2:TRUE LEADING:3 TRAILING:3 SLIDINGWINDOW:4:20 MINLEN:36

Option B: FastP (with HTML/JSON reports)

conda create --name fastp_env
conda activate fastp_env
conda install -c bioconda fastp

Single-end

fastp -i input.fastq -o output.fastq -h report.html -j report.json

Paired-end

fastp -i input_1.fastq -I input_2.fastq -o output_1.fastq -O output_2.fastq -h fastp_report.html -j fastp_report.json

üß¨ 6. Read Mapping & Variant Calling

Tools Setup

conda create --name mapping_vcf
conda activate mapping_vcf
sudo apt-get update && sudo apt-get install -y bwa samtools bcftools

Mapping Workflow

bwa index reference.fasta
# Index reference genome samtools faidx reference.fasta # Create FASTA index
bwa mem reference.fasta R1.fastq R2.fastq > aligned_reads.sam
samtools view -S -b aligned_reads.sam > aligned_reads.bam
samtools sort aligned_reads.bam -o aligned_reads_sorted.bam
samtools index aligned_reads_sorted.bam
samtools flagstat aligned_reads_sorted.bam > mapping_report.txt

Variant Calling with FreeBayes

conda create -n freebayes_env python=3.9 -y
conda activate freebayes_env
conda install -c bioconda freebayes freebayes -f reference.fasta aligned_reads_sorted.bam > variants.vcf

VCF Processing with BCFTools

conda create -n bcftools_env
conda activate bcftools_env
conda install -c bioconda bcftools htslib
bgzip variants.vcf
bcftools index variants.vcf.gz
bcftools filter -i 'QUAL>30' variants.vcf.gz -o high_quality.vcf
bcftools stats variants.vcf.gz > stats.txt

Variant Annotation with SnpEff

conda create --name snpeff_env
conda activate snpeff_env
conda install -c bioconda snpeff
snpeff GRCh37.75 variants.vcf > annotated_variants.vcf

üß© 8. Genome Assembly

SPAdes Assembly

conda create -n spades_env python=3.8 -y
conda activate spades_env
conda install -c bioconda spades

Paired-end

spades.py -1 R1.fastq -2 R2.fastq -o spades_output

Single-End

div class="command">spades.py -s reads.fastq -o spades_output

Hybrid (Illumina + Nanopore)

spades.py -1 reads_1.fastq -2 reads_2.fastq --nanopore nanopore_reads.fastq -o output_dir

With more resources

spades.py -1 reads_1.fastq -2 reads_2.fastq -o spades_output --threads 8 --memory 16

QUAST Quality Assessment

conda create -n quast_ev
conda activate quast_ev
conda install -c bioconda quast
# Basic
quast contigs.fasta -o quast_output
# Comparative
quast assembly1.fasta assembly2.fasta -o quast_output
# With reference
quast contigs.fasta -o quast_output --threads 8 --reference reference_genome.fasta

üåå 11. Galaxy Workflow

1. Quality Control

  1. Upload R1/R2 FASTQ files
  2. Run FastQC on raw reads

2. Read Trimming

Option A: fastp

  • Select paired-end mode
  • Enable adapter auto-detection
  • Enable overrepresented analysis

Option B: Trimmomatic

  • Select paired-end mode
  • Set trimming parameters

3. Assembly (SPAdes)

  • Operation mode: Assembly and error correction
  • K-mer detection: Auto
  • Phred quality offset: Auto

4. Quality Assessment

  • QUAST: Upload contigs.fasta
  • BUSCO: Check genome completeness

5. Annotation (Prokka)

  • Input: contigs.fasta
  • Select kingdom (Bacteria/Virus)
